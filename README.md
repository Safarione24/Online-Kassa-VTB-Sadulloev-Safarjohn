## ТРЕБОВАНИЯ:
python version = 3.13.2
файл 'Vtb-logo1.png'(уже в папке)

## УСТАНОВКА И ЗАПУСК:
__________________________________________________________________
1. Установка зависимостей:
pip install -r requirements.txt

2. Создание тестовых клиентов(если нет data.json или он повреждён):
python data.py

3. Запуск приложения:
python main.py
__________________________________________________________________

## ВАЖНО: 
**НЕ просто запускать файл, а открыть всю папку в VS Code (File → Open Folder) или запускать из терминала, предварительно перейти в саму папку через (cd Online-Kassa-VTB-Sadulloev-Safarjohn)!**

Если ошибка "No module named 'PIL'":
pip install Pillow==12.0.0
